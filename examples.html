Struktur der Module

Jedes Beispiel hat eine eigene HTML-Datei im Ordner examples/ und bindet Module aus assets/js/modules/ ein.

<link rel="stylesheet" href="https://unpkg.com/maplibre-gl@4.1.2/dist/maplibre-gl.css">
<script src="https://unpkg.com/maplibre-gl@4.1.2/dist/maplibre-gl.js"></script>
<script src="https://unpkg.com/pmtiles@2.11.0/dist/index.js"></script>
<script src="../assets/js/modules/base-map.js"></script>

Nächste Module

Lege eine neue HTML-Datei in examples/ an und binde die gewünschten Module aus assets/js/modules/ ein. Die Übersicht lässt sich dann um eine weitere Kachel ergänzen.
